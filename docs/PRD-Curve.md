# 커브 (Curve) - Product Requirements Document

## 1. 제품 개요

### 1.1 제품명
**커브 (Curve)** - 이직 준비 올인원 플랫폼

### 1.2 비전
구직자가 체계적으로 이직을 준비하고, AI 기반 맞춤 이력서를 생성하며, 지원 현황을 효율적으로 관리할 수 있는 모바일 퍼스트 웹 애플리케이션

### 1.3 타겟 사용자
- 경력직 이직 준비생
- 적극적인 구직 활동 중인 직장인
- 다수의 채용공고를 동시에 관리해야 하는 사용자

---

## 2. 핵심 기능

### 2.1 목표 설정 (Goals Tab)
- **이직 목표 수립**: 이직 이유, 커리어 패스, 검색 기간 설정
- **회사 평가 기준 5가지 설정**: 가중치 포함한 개인화된 평가 기준
- **목표 기록 관리**: 시작일/종료일, 결과 기록
- **이전 기록 조회**: 과거 이직 시도 이력 보관

### 2.2 채팅 (Chat Tab)
- **공고 URL 분석**: LinkedIn, 채용사이트 URL 입력 시 AI가 자동 파싱
- **공고 정보 추출**: 회사명, 포지션, 경력요건, 근무형태, 위치, 비자지원 여부
- **핵심 역량 5가지 추출**: 채용담당자 관점에서 중요한 역량 AI 분석
- **중복 URL 감지**: 기존 등록된 공고 재등록 방지

### 2.3 보드 (Board Tab)
- **칸반 보기**: 지원검토 → 서류지원 → 인터뷰 → 오퍼 → 합격-최종
- **테이블 보기**: 스프레드시트 형태의 공고 리스트
- **드래그앤드롭**: 상태 변경을 위한 직관적 UX
- **필터링**: 최소 경력, 근무 형태, 위치 기준 필터
- **정렬**: 추천순(적합도 기반), 최신순, 회사명순(한글→영문)

### 2.4 공고 상세
- **AI 요약**: 공고 핵심 내용 요약
- **우선순위 관리**: 1~5 순위, 회사 매력도+내 적합도 기반 자동 계산
- **회사 평가**: 사용자 설정 5가지 기준으로 별점 평가 (0점 가능)
- **적합도 평가**: AI가 내 경험 기반 핵심 역량별 적합도 분석
- **맞춤 이력서 생성**: 공고별 최적화된 이력서 AI 생성

### 2.5 경력 (Career Tab)
- **이력서 업로드**: PDF 업로드 → AI 파싱 → 경험 자동 추출
- **경력 관리**: 직접 추가/수정/삭제
- **프로젝트 관리**: Selected Projects 섹션
- **공고별 이력서**: 생성된 맞춤 이력서 저장/미리보기/편집/다운로드
- **이력서 내보내기**: DOCX 형식 추출

### 2.6 설정 (Settings Tab)
- **프로필 관리**: 이름 수정
- **비밀번호 변경**: 이메일 링크를 통한 재설정
- **계정 삭제**: 완전 삭제 (auth.users 포함)
- **이용약관/개인정보처리방침**: 법적 문서 제공
- **로그아웃**

## 3. 기술 스택

### 3.1 프론트엔드
- React 18 + TypeScript
- Vite (빌드 도구)
- Tailwind CSS + shadcn/ui
- Zustand (상태관리)
- React Router (라우팅)

### 3.2 백엔드
- Supabase (Lovable Cloud)
  - PostgreSQL Database
  - Edge Functions (Deno)
  - Authentication
  - Row Level Security

### 3.3 AI 통합
- Lovable AI Gateway
  - google/gemini-2.5-flash (기본 모델)
  - 공고 분석, 이력서 생성, 적합도 평가

### 3.4 외부 연동
- Firecrawl API (웹 스크래핑)

---

## 4. 데이터 모델

### 4.1 주요 엔티티
- **profiles**: 사용자 프로필 (user_id, name)
- **JobPosting**: 채용공고 (로컬 스토어)
- **Experience**: 경력/프로젝트 (로컬 스토어)
- **TailoredResume**: 공고별 맞춤 이력서 (로컬 스토어)
- **CareerGoal**: 이직 목표 (로컬 스토어)
- **ChatMessage**: 채팅 기록 (로컬 스토어)

---

## 5. 사용자 플로우

```
1. 회원가입/로그인
     ↓
2. 이직 목표 설정 (Goals)
     ↓
3. 이력서 업로드 (Career)
     ↓
4. 채용공고 URL 입력 (Chat)
     ↓
5. 공고 분석 및 보드 등록
     ↓
6. 회사 평가 & 적합도 분석 (Board → Detail)
     ↓
7. 맞춤 이력서 생성
     ↓
8. 지원 및 상태 관리
```
